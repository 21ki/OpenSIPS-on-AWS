#How to run a file
* Download files to your server

```

# git clone https://github.com/chanon-m/opensips-on-aws.git

```

##OpenSIPS

* Copy opensips.cfg to /etc/opensips

```

# cp ./opensips-on-aws/opensips.cfg /etc/opensips

```

* Restart OpenSIPS
